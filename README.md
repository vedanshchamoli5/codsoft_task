## task 2
Lumina — Creative Studio Landing Page
A fully responsive, production-grade landing page built with pure HTML and CSS (plus a touch of vanilla JavaScript). Designed as a premium creative studio website with a warm editorial aesthetic, smooth animations, and clean layout structure.

🖥️ Live Preview

Open landing-page.html directly in any browser — no build tools or dependencies required.


📁 Project Structure
lumina-landing-page/
│
├── landing-page.html   # Single-file project (HTML + CSS + JS)
└── README.md

✨ Features

Sticky Navigation — fixed top bar with blur backdrop and smooth scroll links
Hero Section — two-column layout with headline, stats, CTA buttons, and an animated decorative panel
Marquee Strip — seamless infinite-scroll ticker banner
Services Grid — 6-card feature section with hover underline animations
Works Showcase — dark asymmetric project grid with hover reveal effects
Testimonials — 3-column client review cards with lift-on-hover
Email CTA — newsletter/contact form with live input validation
Footer — 4-column layout with social links, site map, and legal links
Scroll Reveal Animations — elements fade in as you scroll using IntersectionObserver
Fully Responsive — mobile-friendly layout via CSS media queries


🎨 Design System
TokenValueBackground#f5f0e8 (warm cream)Foreground#0d0d0d (deep ink)Accent#c8a84b (gold)Highlight#c0502a (rust)Muted Text#6b6560Display FontPlayfair Display (Google Fonts)Body FontDM Sans (Google Fonts)

🛠️ Built With

HTML5 — semantic markup
CSS3 — custom properties, grid, flexbox, keyframe animations, backdrop-filter
Vanilla JavaScript — IntersectionObserver for scroll reveals, form validation
Google Fonts — Playfair Display & DM Sans

No frameworks. No build step. No dependencies.
